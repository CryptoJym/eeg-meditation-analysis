## Accessing Raw Data

The raw CSV files are available locally but not synced to GitHub due to file size constraints.
For data access, please contact the repository owner or run analyses locally with the provided scripts.

## File Format

Both CSV files contain:
- Timestamp column
- Single-channel EEG data (256 Hz sampling rate)
- Raw voltage readings from the Research Kit device
