## Technical Notes

### Recording Parameters
- **Device**: Research Kit EEG System
- **Sampling Rate**: 256 Hz
- **Channels**: Single channel recordings
- **Filtering**: 0.5-50 Hz bandpass applied

### Analysis Methods
- Welch's method for power spectral density
- Sliding window analysis (30-second windows, 50% overlap)
- Artifact removal using statistical thresholding
- Frequency band extraction using butterworth filters

### Quality Indicators
- **Signal Quality**: Excellent (minimal artifacts)
- **Recording Stability**: Consistent throughout sessions
- **Data Completeness**: 100% for both sessions
